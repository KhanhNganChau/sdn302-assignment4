This is a [React](https://react.dev) project powered by [Vite](https://vite.dev).

## Getting Started

From the workspace root, run the development server for this app:

```bash
pnpm --filter web dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/App.tsx`. The page auto-updates as you edit the file.

## Available Scripts

```bash
pnpm --filter web dev

pnpm --filter web preview

```
